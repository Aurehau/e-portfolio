Site | Escape game à Mulhouse 2024

Description

En groupe de 2, nous devions faire un escape game à Mulhouse sous forme de site. L'objectif est de visiter la ville en faisant des jeux sur le téléphone. J'ai fait 2 des jeux sur le site ainsi que la majorité de l'histoire entre les jeux.
Pour les jeux, j'ai fait un labyrinthe dont l'objectif est de trouver un coffre. Une fois le coffre trouvé, on arrive dans la pièce reproduite en 3D, où on va trouver des indices pour ouvrir le coffre. Toutes les animations des personnages sont faites en css.

Pour faire le labyrinthe, j'ai désinné le décor sur Figma, ainsi que le personnage qui se déplasse. Le personnage se dirige vers la position de la souris et les murs sont matérialisés par des tests de position sur les axes X et Y. Si la prochaine position est 30 X et 5 Y mais qu'elle correspond à un mur alors le personnage n'avancera pas dans cette direction.
En fonction de la taille de l'écran, le labyrinthe peut dépasser. Si la position suivante est hors de l'écran, tout le labyrinthe est déplacé pour rester dans l'écran.
Pour le 2ème jeu, j'ai reproduit la pièce, où se trouve le coffre à trouver, en 3d avec 3ds max. J'ai fait la texture, sur la quelle il y a les indices, avec figma. j'ai ensuite mis le décors 3d sur le site web grâce à A-Frame.
J'ai utilisé Figma pour dessiner tous les éléments, donc le personnage principal, les différents objets et autres illustrations. J'ai ensuite animé le personnage avec du css, j'ai notamment reproduit les yeux du personnage avec des div pour l'animer comme je le souhaitais.

On peut aussi noter qu'il y a une carte qui indique le chemin vers les lieux où se rendre pour visiter Mulhouse et faire les jeux.
